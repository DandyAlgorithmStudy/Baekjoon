## 진행 방법

0. 문제집: solved.ac Class3 ~ Class4
1. 공통 문제 N 문제를 고른다.
2. 개인 발표 문제 (N/4) 문제를 분배한다.
3. 매주 금요일 전까지 공통 문제 N 문제를 전부 풀고 리포지토리에 업로드 한다.
4. 매주 금요일 개인 발표 문제 (N/4) 문제를 어떠한 방식으로 풀었는지 멤버들에게 발표한다.
5. 1번부터 다시 반복한다.
